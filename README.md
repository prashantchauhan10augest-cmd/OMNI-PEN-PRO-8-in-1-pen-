Omni Pen Pro - 8-in-1 3D Visualization

An interactive, web-based 3D mechanical feasibility model of the OmniPen Pro, a premium 8-in-1 multifunction tactical writing instrument. This project visualizes the product's internal architecture, rotary selection mechanism, and "Tactical Black & Chrome" aesthetic using WebGL.
